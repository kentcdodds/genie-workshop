This is the initial setup for the Genie-Workshop. Here are a few things of interest before you begin:
  • Even though this workshop uses AngularJS, genie itself has no dependencies. However, ux-genie is an AngularJS module and therefore depends on AngularJS.
  • The only stuff you really need to concern yourself with for the workshop are the seconds of code that exist in between the commented lines.
  • The section above will show you what concept you're learning, its branch, and the next concept/branch. When you're finished with this section, just execute that git command.
  • This section will contain information about the concept.
  • The section below will illustrate the concept.
The process to setup your project to use genie is pretty simple. Simply insert a script tag before your </body> tag with the source to your genie file. Like so: 
<script src="/vendor/genie.min.js"></script>
*If you're using ux-genie, you'll want to do the same with ux-genie, except make sure you include it before you load your own module.
